# 综述

墨国：等级法律森严是因为人口多。人与人素质落差大（因为国土大，交通不便）

奥国： 主权掌握在贵族手上，贵族建立的教会是国家的基本结构，服务性公务由皇室承担。

沙国： 自由主义。但是国力不行。

# 沙弥娜

沙国教合会（最高权力机构），实际权力也不大，掌管为人民服务的事务。
